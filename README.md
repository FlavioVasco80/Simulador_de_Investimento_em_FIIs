Simulador_de_Investimento_em_FIIs 

DESCRIÇÃO DO PROJETO

Ferramenta desenvolvida para o desafio de projeto da DIO, com o objetivo de simular a evolução patrimonial e o recebimento de dividendos em Fundos Imobiliários. 

CONTEXTO PROFISSIONAL 

Como profissional formado em Gestão Logística e Processos Gerenciais, utilizei esse projeto para aplicar conceitos de organização de dados, automação de cálculos e visão estratégica financeira, competências essenciais para a tomada de decisão. 

FUNCIONALIDADES DA PLANILHA 

Configurações Dinâmicas: Permite definir o salário, rendimanto atual da carteira e sugere automaticamente o valor de investimento com base na capacidade financeira. 

Análise de Cenários Temporais: Projeção detalhada de patrimônio e dividendos para períodos de 2, 5, 10, 20, e até 30 anos. 

Seleção de Perfil de Investidor: Menu interativo para escolher entre os perfis Conservador, Moderado ou Agressivo, que altera automaticamente a sugestão de alocação de ativos. 

Alocação Estratégica de Ativos (Asset Allocation): Distribuição percentual sugerida entre diferentes tipos de FIIs (Papel, Tijolo, Hibrídos, etc.) representada por um gráfico de pizza dinâmico. 

TECNOLOGIAS UTILIZADAS 

Microsoft Excel: 
Fórmulas financeiras matemáticas (para os cálculos de juros compostos e dividendos); 
Validação de dados (para a criação de menus suspensos onde você seleciona o perfil de investidor); 
Gráficos dinâmicos (para a representação visual da alocação de ativos); 
Desing de Dashboard/ UX (para usabilidade, intuição, hierarquia visual e acessibilidade). 

Git/GitHub: Para o controle da versão e documentação (README) do projeto.
